# 🚀 LLM Zero to Hero



## 📖 关于本项目

本仓库包含了llm相关技术，主要用于练手。

---

## ✅ 已实现

### 核心架构
- **🔷 Transformer** - 基于注意力机制的基础架构
  - 多头自注意力机制
  - 位置编码
  - 前馈神经网络

- **🦙 LLaMA 2** - Meta的开源语言模型
  - 旋转位置编码（RoPE）
  - SwiGLU激活函数
  - 分组查询注意力

### 训练与微调
- **⚡ LoRA** (低秩适应) - 高效微调技术
  - 参数高效的迁移学习
  - 适配器模块
  - 秩分解

- **🎯 PPO** (近端策略优化) - 基于人类反馈的强化学习
  - Actor-Critic架构
  - 策略梯度优化
  - 裁剪代理目标
  - 详细文档：[PPO算法详解](PPO/PPO算法详解.md)
